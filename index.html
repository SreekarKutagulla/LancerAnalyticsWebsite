What do we do?

Formation and Play Analysis

Collect source data from HUDL and analyze formation frequency and formation situational efficiency. Designed and implemented a formation efficiency search system + GUI as well.

Player Profiles

Analyze game statistics and practice film in order to create player profiles, optimal lineups and scheme fits.

Scouting

Source data from HUDL and run Python & R scripts to create clear visualizations on opponents play.

Reverse Scouting

Source data from HUDL and analyze practice film to analyze team's efficiency against specifc opposing formations.
